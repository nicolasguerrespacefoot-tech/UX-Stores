Outil interne boutique

Assistant Boutique — Aide à la réponse client

Cet outil accompagne les collaborateurs boutique lorsqu’un client appelle au sujet d’une commande passée sur le site internet. Il permet d’identifier rapidement la situation, de poser les bonnes questions, d’utiliser une réponse homogène et d’orienter le client vers le bon service si nécessaire.

Pourquoi cet outil ?

Le contexte

Les boutiques reçoivent de plus en plus d’appels de clients ayant passé une commande web. Les demandes concernent souvent le suivi de livraison, les retards, les colis indiqués livrés mais non reçus, les retours, les remboursements ou les réclamations produit.

Client
Il veut une réponse immédiate, même si sa commande vient du web.
Boutique
Elle doit rassurer, qualifier et orienter sans surpromettre.
SAV web
Il garde la résolution des cas complexes : transport, remboursement, litige.
Mode d’emploi

Comment utiliser l’assistant pendant un appel ?

1. Identifier le motif

Sélectionner le cas qui correspond à la demande du client : retard, colis non reçu, retour, remboursement, etc.

2. Poser les questions

Répondre aux questions oui/non pour qualifier la situation sans oublier d’étape importante.

3. Lire la réponse

Utiliser le script proposé pour garder une formulation claire, rassurante et homogène.

4. Appliquer l’action

Vérifier l’information, orienter vers le SAV, escalader ou clôturer selon le résultat affiché.

Posture client

La règle de réponse en boutique

À faire
  • Écouter le client sans l’interrompre.
  • Reformuler simplement sa demande.
  • Vérifier les informations disponibles.
  • Expliquer la prochaine étape.
  • Orienter vers le SAV si le sujet dépasse le rôle boutique.
À éviter
  • Promettre un remboursement sans validation.
  • Accuser le transporteur ou l’entrepôt.
  • Dire “je ne peux rien faire”.
  • Créer plusieurs demandes pour le même sujet.
  • Donner un délai précis non confirmé.
Rôles & limites

Ce que la boutique peut faire ou non

Boutique

  • Accueillir et rassurer le client.
  • Qualifier la demande.
  • Vérifier les informations visibles dans les outils autorisés.
  • Lire le script adapté.
  • Orienter vers le bon canal SAV.

SAV web

  • Gérer les litiges transport.
  • Ouvrir les enquêtes transporteur.
  • Valider les remboursements web.
  • Traiter les réclamations produit.
  • Gérer les escalades sensibles.
