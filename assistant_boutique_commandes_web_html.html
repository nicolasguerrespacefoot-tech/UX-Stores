Posture client

Les règles de communication à respecter lors de chaque appel entrant.

À faire
Écouter le client jusqu'au bout sans l'interrompre.
Reformuler simplement sa demande pour confirmer la compréhension.
Vérifier les informations disponibles dans les outils autorisés.
Expliquer clairement la prochaine étape.
Orienter vers le SAV si le sujet dépasse le périmètre boutique.
À éviter
Promettre un remboursement sans validation du SAV.
Accuser le transporteur ou l'entrepôt directement.
Dire "je ne peux rien faire" sans orienter vers une solution.
Créer plusieurs demandes pour le même sujet client.
Donner un délai précis non confirmé par les systèmes.

Référence

Rôles & limites

Ce que la boutique peut faire, et ce qui relève du SAV web.

Boutique
  • Accueillir et rassurer le client.
  • Qualifier la demande via le diagnostic.
  • Vérifier les informations dans les outils autorisés.
  • Lire le script adapté à la situation.
  • Orienter vers le bon canal SAV.
  • Copier et envoyer le paquet de transfert par Teams.
SAV Web
  • Gérer les litiges transport.
  • Ouvrir les enquêtes transporteur.
  • Valider les remboursements web.
  • Traiter les réclamations produit.
  • Gérer les escalades sensibles.
Contexte
Les boutiques reçoivent de plus en plus d'appels de clients ayant passé une commande web. La boutique n'a pas accès aux mêmes outils que le SAV web et ne peut pas ouvrir de litiges transport ou valider des remboursements. Son rôle est d'écouter, qualifier, orienter et transmettre les informations au SAV via Teams pour assurer une prise en charge rapide.
